This is the Mobile Application for [Geo Smart System](https://github.com/supanadit/geosmartsystem) and [Geo Smart Map](https://github.com/supanadit/geosmartmap)

## Requirements
- Flutter
- Android 6.0.1+

## Quick Start
- Clone this repository
- `flutter pub get`
- `flutter run`
